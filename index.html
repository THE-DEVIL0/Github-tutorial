Git and Github

Git

Git is a version control system that lets you keep track of changes in code .

Git is :
  • popular
  • free and open source
  • fast and scalable
  • used for Collaboration

GitHub

A Website that allows users to store and manage their code using Git .

  • Visual studio code

Configuring Git

Using Git bash or terminal write the following commands to configure git.

  • git config --global user.name"Your name"
  • git config --global user.email "your_email@example.com"
  • git config --list

Git Commands for VScode

  1. Clone command
    • Use this command to clone your repository into local system.
    • Write git clone and then paste the https link you copied earlier.
  2. cd command
    • Use this command to move to your desired folder
    • Writecd and then the path of your folder
  3. ls-a or ls-hidden
    • Use this command to know if your folder is conneced to Github or not
    • If it shows .git at last that means it's connected otherwise not.
  4. status command
    • Use this command to know the status of your code
    • It can be one of the following
      • Untracked i.e new files that git does'nt know about
      • Modified i.e a change in a tracked file
      • Staged i.e change is ready to be commited
      • Unmodified i.e uncahanged

Add and Commit

Add

Use this command to add new or changed files to git staging area


In order to do this just write git add .

Commit

Use this command to add a record of the change


In order to do this just write git commit -m "Some meassage"

Push

Use this command to push the changes to your github repository


In order to do this just write git push origin main
